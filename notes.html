Getting Started with HTML

The World Wide Web

The World Wide Web is the largest network of computers. It consists of a computer and a browser(both are referred to as the client), the cloud(Internet) and the server. Communication between the client and the server takes the Hypertext Transfer Protocol(HTTP). A request for a document is placed to the server using the browser. Interraction is done through HTTP and the request gets delivered to the server using the internet. The requested file(s), an HTML document is sent back to the client, through the internet and rendered using the bowser. The document for the World Wide Web is the HyperText Markup language(HTML). Other files such as images, audio, videos, pdf etc can be transferred over the internet.

HTML Elements

eg.<p> This is fun </p>

An HTML element consists of:

  • An opening tag
  • A content
  • A closing tag

From the example above, <p> is an opening tag, "This is fun" is the content and </p> is the closing tag.

Exceptions

There are a group of elements called the void elements. These elements do not follow the earlier rule of the composition of an element. They do have opening tags but no closing tags. An example is the <img>, <br> tags.

HTML Attributes

Again, for some elements such as <img src = "URL"> it has an attribute "src" with its value as the" URL" of the image which is to be added to the HTML document. Another example is the anchor tag link. The "href" attribute has a value of "URL". When the HTML document is rendered in the browser, when the pointer hovers over "link" , it shows its a clickable link.

HTML Structure

The structure of HTML is tree like. It is written as follows

               
<!DOCTYPE html>
<html>
<head>
   <title> document title </title>
</head>
<body>
       content goes here
</body>
</html>

Examples of HTML Elements

<p> : the <p>( used <p> Content<p>) element is a Block element that defines a paragraph. Texts in the tag are contained in an invisible box with defined width and height.

<br>: Unlike <p>, <br> is an inline element. It creates a line break at the point of insertion.

Container Elements

<span> content<span> and <div>content<div> are container elements. The difference being that <span> is an inline element and <div> is a block element. They both facilitate the addition of Cascading Style Sheet to an element(s).
